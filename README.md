# Taekyung Park Portfolio Homepage

`dukgooenv.github.io`의 큰 틀을 참고해 `Taekyung Park` 포트폴리오 메뉴와 작품 분류를 재구성한 GitHub Pages용 정적 홈페이지입니다.

## 현재 메뉴 구성

- `All portfolio`: 전체 작품
- `About`: 소개, 언어, 경력, 연락처
- `Professional`: Webzen Project N, Rootnstudio WHITEDAY2
- `Personal`: 개인 작업 전체

## 수정할 곳

- `index.html`: 소개 문구, 경력 설명, 작품 제목, 작품 이미지, 이메일을 바꾸세요.
- `style.css`: 색상, 간격, 글꼴, 반응형 레이아웃을 바꾸세요.
- `script.js`: 카테고리 필터와 이메일 복사 동작이 들어 있습니다.

## GitHub Pages 배포

1. GitHub에서 `username.github.io` 저장소를 만듭니다.
2. 이 폴더의 `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드합니다.
3. 저장소의 `Settings` → `Pages`에서 `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/root` 폴더를 선택하면 `https://username.github.io/` 주소로 열립니다.
